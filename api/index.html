Class Reference

Class Description
macgyer\yii2materializecss\assets\MaterializeAsset MaterializeAsset provides the required Materialize CSS files.
macgyer\yii2materializecss\assets\MaterializeFontAsset MaterializeFontAsset provides the Material Icons font family CSS.
macgyer\yii2materializecss\assets\MaterializePluginAsset MaterializePluginAsset provides the Materialize JS files.
macgyer\yii2materializecss\assets\NoUiSliderAsset The JS plugin noUiSlider adds useful functionality for slider (i. e. range) input elements.
macgyer\yii2materializecss\lib\BaseInputWidget BaseInputWidget is the base class for all input widgets in this extension.
macgyer\yii2materializecss\lib\BaseWidget BaseWidget is the base class for all non-input widgets in this extension.
macgyer\yii2materializecss\lib\Html Html is the individual HTML helper implementation.
macgyer\yii2materializecss\lib\MaterializeWidgetTrait MaterializeWidgetTrait provides the basics for all Materialize widgets features.
macgyer\yii2materializecss\widgets\Alert Alert renders Yii's session flash messages.
macgyer\yii2materializecss\widgets\Button Button renders an HTML button.
macgyer\yii2materializecss\widgets\Collapsible Collapsibles are accordion elements that expand when clicked on. They allow you to hide content that is not immediately relevant to the user.
macgyer\yii2materializecss\widgets\Icon Icon can be used to display a Materialize icon.
macgyer\yii2materializecss\widgets\Modal Modal renders a modal window that can be toggled by clicking on a button.
macgyer\yii2materializecss\widgets\Progress Progress renders a linear progress bar.
macgyer\yii2materializecss\widgets\Spinner Spinner renders a circular loading animation.
macgyer\yii2materializecss\widgets\StaticChip StaticChip renders small units of information.
macgyer\yii2materializecss\widgets\data\DetailView DetailView displays the detail of a single data model.
macgyer\yii2materializecss\widgets\form\ActiveField ActiveField represents a form input field within an yii\widgets\ActiveForm.
macgyer\yii2materializecss\widgets\form\ActiveForm ActiveForm is a widget that builds an interactive HTML form for one or multiple data models.
macgyer\yii2materializecss\widgets\form\ChipInput ChipInput implements the Materialize Chips JS plugin functionality.
macgyer\yii2materializecss\widgets\form\DatePicker DatePicker renders an date picker input element.
macgyer\yii2materializecss\widgets\form\RangeInput RangeInput renders a slider input element. You can specify to use a simple HTML5 range <input> or make use of the versatile noUiSlider JS plugin.
macgyer\yii2materializecss\widgets\form\Select Select renders a <select> option list and utilizes Materialize material_select() to display a highly user-friendly drop down.
macgyer\yii2materializecss\widgets\form\SubmitButton SubmitButton renders an HTML submit button.
macgyer\yii2materializecss\widgets\form\SwitchButton SwitchButton renders a toggle button.
macgyer\yii2materializecss\widgets\form\TimePicker TimePicker renders a time picker input element with a modified version of pickadate.js.
macgyer\yii2materializecss\widgets\grid\ActionColumn ActionColumn is a column for the GridView widget that displays buttons for viewing and manipulating the items.
macgyer\yii2materializecss\widgets\grid\GridView The GridView widget is used to display data in a grid.
macgyer\yii2materializecss\widgets\grid\LinkPager LinkPager displays a list of hyperlinks that lead to different pages of target.
macgyer\yii2materializecss\widgets\media\Carousel Carousel is a robust and versatile component that can be an image slider or an item carousel with arbitrary HTML content.
macgyer\yii2materializecss\widgets\media\MaterialBox MaterialBox creates a lightweight lightbox variant to present images.
macgyer\yii2materializecss\widgets\media\Parallax Parallax renders an image container where the background image is scrolled at another speed than the foreground.
macgyer\yii2materializecss\widgets\media\Slider Slider renders a Materialize image slider with optional captions.
macgyer\yii2materializecss\widgets\navigation\Breadcrumbs Breadcrumbs displays a list of links indicating the position of the current page in the whole site hierarchy.
macgyer\yii2materializecss\widgets\navigation\Dropdown Dropdown renders a Materialize dropdown menu component.
macgyer\yii2materializecss\widgets\navigation\FloatingActionButton Floating action buttons (FAB) are sets of actions grouped by a single button.
macgyer\yii2materializecss\widgets\navigation\Nav Nav renders a navigation items list.
macgyer\yii2materializecss\widgets\navigation\NavBar NavBar renders a navbar HTML component.
macgyer\yii2materializecss\widgets\navigation\SideNav SideNav renders a side navigation, which is especially useful for mobile devices or small screen sizes.
