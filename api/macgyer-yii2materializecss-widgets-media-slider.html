Class macgyer\yii2materializecss\widgets\media\Slider

All Classes | Properties | Methods | Constants
Inheritancemacgyer\yii2materializecss\widgets\media\Slider » macgyer\yii2materializecss\lib\BaseWidget » yii\base\Widget
Uses Traitsmacgyer\yii2materializecss\lib\MaterializeWidgetTrait

Slider renders a Materialize image slider with optional captions.

Simply provide the $slides as an array of items. For each item you must define the image key with the image's src. Additionally you can define and align a caption for every slide individually. Caption content can be HTML and will not be encoded.

'slideOptions' => [
     'class' => 'slide-item' // this class will be used for all slide elements (<li>)
],
'slides' => [
     [
         'image' => ['src' => '/source/of/image'],
     ],
     [
         'image' => ['src' => '/source/of/image'],
         'caption' => [
             'content' => '<p>Caption content</p>',
             'align' => Slider::CAPTION_ALIGN_RIGHT
         ],
         'options' => ['class' => 'slide-item-override'] // overrides $slideOptions
     ]
]

See also http://nextmaterializecss.com/media.html#slider.
